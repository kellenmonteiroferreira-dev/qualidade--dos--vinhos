#  Análise de Dados: Qualidade dos Vinhos

##  Descrição do Projeto
Este projeto apresenta uma análise exploratória de dados sobre a qualidade de vinhos.  
O objetivo é investigar como características físico-químicas dos vinhos podem influenciar sua qualidade.

A análise foi realizada utilizando Python e bibliotecas voltadas para manipulação e exploração de dados.

---

##  Objetivo da Análise

- Explorar o conjunto de dados de qualidade dos vinhos
- Compreender as variáveis presentes no dataset
- Realizar tratamento e organização dos dados
- Identificar padrões relacionados à qualidade dos vinhos

---

##  Dataset

O conjunto de dados contém informações físico-químicas dos vinhos, como:

- acidez
- açúcar residual
- pH
- teor alcoólico
- densidade
- qualidade do vinho

Essas variáveis permitem analisar possíveis relações entre as características químicas e a qualidade final do vinho.
